# V0-Umbilical-Klicky
Voron V0 / V0.1 Umbilical remix for klicky probe

This is remix of timmit99 Umbilical cable mod for the Voron V0.1, I have chanegd the design to add in the z-end stop of klicky probe function as I want a clean look, I may later add the I2C wiring for the ADXL345 for input shaping.

The bill of materials is the same as timmit99 but with a 16 pin Molex not a 14 pin.
